# Section 2: Security Analysis

#### 1. Shannon Entropy
Each bit doubles the effort for an attacker (Shannon, 1948).

#### 2. Cracking Time
For 37.6 bits at 10^9 attempts/s: 3.42 minutes.

#### 3. Theoretical vs Practical
Patterns like 'P@ssw0rd' decrease actual entropy (Smith, 2023).
