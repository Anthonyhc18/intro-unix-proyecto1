# Intro to UNIX - Project 1

**Student:** Anthony Herrera, Francisco Muela, Luis Tenemaza 

 

## Project Description

Implementation of security protocols in Debian 13, including LUKS disk encryption, password entropy analysis using Shannon's formula, hash verification with Hashcat, and secure communication via GPG and ProtonMail.

 

## Repository Contents

* **section1/:** Installation and Disk Encryption evidence.

* **section2/:** Entropy calculations and KeePassXC.

* **section3/:** Hash integrity and Hashcat cracking results.

* **section4/:** Privacy reflection and GPG keys.
